📘 Tutorial Pemakaian

1. Upload tools ini ke hosting (Netlify, Vercel, dll).

2. Hubungkan dengan website kamu: website kamu akan mengirim data ke engine ini.

3. Di website kamu, buat tampilan sendiri (isi pesan, baterai, sinyal, jam, bahasa).

4. Kirim pengaturan ke tools ini (via JavaScript).

5. Proses generate otomatis: tools ini akan menyesuaikan tampilan.

6. Ambil hasil gambar: screenshot dikirim kembali ke website kamu.

Pastikan tools online, koneksi benar, dan jangan ubah sistem utama.
